# Simple Golf Game

A web-based golf game inspired by Forest Golf Planner. Plan your shots carefully to navigate obstacles and reach the hole in as few strokes as possible!

## Features

- 9 unique holes with varying difficulty
- Multiple obstacle types:
  - **Trees**: Bounce off them
  - **Rocks**: Solid obstacles that deflect the ball
  - **Water hazards**: Reset your ball position if you land in them
- Power and angle controls for precise shot planning
- Trajectory preview to help plan your shots
- Par scoring system
- Progressive difficulty

## How to Play

1. **Aim**: Use the angle slider to set your shot direction (0-360 degrees)
2. **Power**: Use the power slider to set shot strength (10-100%)
3. **Preview**: A dotted line shows your planned trajectory
4. **Shoot**: Click the "Shoot!" button to take your shot
5. **Goal**: Get the ball into the black hole in as few strokes as possible

## Controls

- **Power Slider**: Adjust shot power from 10% to 100%
- **Angle Slider**: Set shot direction from 0° to 360°
- **Shoot Button**: Execute your shot
- **Reset Ball**: Return ball to starting position (doesn't count as stroke)
- **Next Hole**: Proceed to the next hole after completing current one

## Obstacles

- **Trees** (Green circles with brown trunks): Bounce off but lose momentum
- **Rocks** (Gray rectangles): Solid barriers that deflect shots
- **Water** (Blue animated areas): Ball resets if it lands here

## Scoring

Each hole has a par value (2-6 strokes). Try to complete each hole in par or better!

## Getting Started

Simply open `index.html` in a web browser to start playing. No installation required!
